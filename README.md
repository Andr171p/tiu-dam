# DAM система

## Модуль импорта

Загрузка медиа через drag-and-drop, проверка на дубликаты (хэширование), обязательное заполнение метаданных (источник, автор, тип лицензии).

## Работа с изображениями

 - <i>Ресайз</i>: Пресеты под соцсети (VK, TG), сайт ТИУ, печать.
 - <i>Водяные знаки</i>: Наложение логотипа ТИУ с выбором прозрачности и позиции.
 - <i>Редактор</i>: Базовая обрезка (crop), поворот, цветокоррекция (через Canvas API).
 - <i>Поиск</i>: По тегам, дате, автору и содержимому (автогенерация тегов используя легкие оптимизированные под CPU модели) + RAG по картинкам (CLIP модели).
 - <i>Журнал использования</i>: кто и когда скачал изображение.

## Отправка медиа

 - Просмотр/скачивание изображения с нужным размером по ссылке (скачивается сразу с водяным знаком). Если сервис равернуть на публичном Web-сервере, то легко интегрировать медиа на сайт через URL-адрес картинки.

## Регистрация и авторизация

Основные варианты:

 1. Кастомная аутентификация через email + пароль (регистрация через приглашение на почту)
 2. Интеграция с SSO ТИУ -> сотрудники смогут входить под своими учётками
